WooCommere Invoice Me Payment Gateway Extends WooCommerce Payment Gateway allow selected customer to checkout without payment.

== Description ==

WooCommerce Checkout without making payment rather select Invoice Me to make manual payment or payment later for known customers. This option is available only for selected and logged in customers.

Pro Version Features
* Allow user role to set invoice me checkout. So all user of that user role can checkout using invoice me gateway.
* Invoice Mark as Paid. See screenshot below for pro features.

[Pro version](https://codecanyon.net/item/woocommere-invoice-me-for-selected-customers/9360130?ref=RMweblab) is available [here](https://codecanyon.net/item/woocommere-invoice-me-for-selected-customers/9360130?ref=RMweblab)

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/woo-invoice-me` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Enable this gateway from WooCommerce settings /wp-admin/admin.php?page=wc-settings&tab=checkout&section=invoiceme


== Frequently Asked Questions ==


= Will it work for all user? =
No, this plugin only work for selected customers/users. Need to allow from user edit page.
